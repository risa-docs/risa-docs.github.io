◆ get_implemented_processors()

std::vector< std::vector< std::string > > get_implemented_processors ( )

Returns a list of implemented processor stages.

This function must be defined with extern "C" to prevent C++ name-mangeling. Each entry of the returned vector is a vector of strings itself, containing the stage name, encoded input type and encoded output type. Type encoding uses a memory space identifier ('c' for CUDA, 'h' for host) followed by the actual type 'T' which can be any plane-old-datatype. E.g. cfloat to encode a float array in device memory or huint16_t for a huint16_t array in host memory.

Return values
Listof processor stages as a vector of entries.
